→ Frontend  
→ Backend  
→ Database  
→ REST API  
→ Admin Dashboard

===========================
   PROJECT OVERVIEW
===========================
We need a modern, fully responsive website for our university club.

The project must include:
- Home Page
- Founding Members Page
- Club Administration Page
- Members Page
- Projects & Activities Page
- Join Us Form
- Contact Page
- Admin Login + Dashboard (with CRUD)
- Database integration

Use any stack you prefer (Node.js + Express + SQLite/PostgreSQL recommended).

===========================
     DATABASE STRUCTURE
===========================

Create these tables:

Table: founders  
- id (PK)  
- full_name  
- role  
- photo_url  
- description  

Table: administration  
- id (PK)  
- full_name  
- role  
- department  
- photo_url  
- description  

Table: members  
- id (PK)  
- full_name  
- specialty  
- study_year  
- photo_url  

Table: projects  
- id (PK)  
- title  
- date  
- description  
- banner_url  

Table: join_requests  
- id (PK)  
- full_name  
- email  
- phone  
- field_of_study  
- study_year  
- motivation  
- created_at  

Also create an admin_users table:
- id  
- username  
- password (hashed)

===========================
     BACKEND REQUIREMENTS
===========================
Create a REST API with CRUD for all tables:
- GET /items
- GET /items/:id
- POST /items
- PUT /items/:id
- DELETE /items/:id

Also:
- Admin login route  
- Middleware for protected routes  
- File upload support for images (optional)
